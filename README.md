## 🎯 项目特色

### 双分数体系

- **近期分数 (score_near)**: 评估当前至2027年的实际 AI 渗透情况
- **理论上限 (score_ceiling)**: 评估5-10年内技术可行性的上限
- **分数差距 (Gap)**: 揭示"技术可行"与"实际部署"之间的差距

### 高信息密度可视化

- **Impact Mode 标签**：替代型/增强型/混合型/对抗型（4色编码）
- **就业影响路径**：直接裁员、任务重组、入职收窄、需求增长
- **政策影响标注**：黄色高亮显示中国特色政策影响
- **新兴职业标记**：AI 训练师、数据标注员等

## 📈 数据来源

### 1. 职业分类体系

- **来源**：《中华人民共和国职业分类大典（2022年版）》
- **使用内容**：
  - 职业名称与编码（8大类 → 75中类 → 434小类 → 1639细类）
  - 职业定义与主要工作任务
  - 技能要求与教育背景
- **PDF 来源**：`职业大典体系分类表.pdf`

### 2. 就业人数数据

- **主要来源**：国家统计局《2024年中国统计年鉴》
- **数据类型**：
  - 分行业就业人员数（19个国民经济行业）
  - 城镇单位就业人员数
  - 分职业就业人员抽样调查数据
- **估算方法**：
  ```
  职业就业量 = 所属行业总就业量 × 职业占比估算系数

  示例：
  - 软件工程师 = 信息传输、软件和信息技术服务业就业量 × 占比估算 ≈ 697.3万
  ```

### 3. 薪资数据

- **来源**：
  - 国家统计局：按行业分城镇单位就业人员平均工资
  - 智联招聘、前程无忧等招聘网站公开数据
  - 行业协会报告（如中国软件行业协会）
- **数据处理**：
  - 优先使用国家统计局官方数据
  - 互联网数据用于细分职业补充
  - 所有薪资数据标准化为"年薪（元）"

### 4. 职业URL（权威来源）

每个职业配备权威参考链接，便于深入了解：

- 国家统计局官网
- 行业协会官方网站
- 人社部官网
- 教育部官网

## 🧮 AI 暴露度计算方法

### 评分流程

#### 第一步：基准评分（参考 Eloundou et al. 2023）

使用 Claude Opus 4.6 评估职业任务的 AI 暴露度：

**评分标准（0-10分）**：

- **0-2分**：AI 几乎无影响（体力劳动、需物理在场）
- **3-4分**：AI 轻度辅助（部分任务可增强）
- **5-6分**：AI 中度影响（核心任务可自动化但需人类监督）
- **7-8分**：AI 高度影响（多数任务可自动化）
- **9-10分**：AI 完全替代（技术上可完全自动化）

**Prompt 模板**：

```
你是劳动经济学职业AI影响评估专家。请评估以下职业的AI暴露度：

职业名称：{occupation_name}
主要任务：{main_tasks}
技能要求：{skills}

请按以下维度评分（0-10）：
1. score_near（2024-2027近期影响）：考虑当前技术成熟度、部署成本、政策环境
2. score_ceiling（5-10年理论上限）：假设技术完全成熟、成本降至可接受水平

输出格式：
{
  "score_near": X,
  "score_ceiling": Y,
  "reasoning": "详细理由...",
  "tasks_at_risk": ["任务1", "任务2"],
  "impact_mode": "替代型/增强型/混合型/对抗型"
}
```

#### 第三步：置信度标注

- **H (High)**：基于权威研究或行业报告（如金税四期、智能客服渗透率）
- **M (Medium)**：基于行业专家访谈或公开数据推算
- **L (Low)**：基于AI模型推理或跨国数据类比

### 数据验证

**交叉验证机制**：

1. 与 Minimax 评分对比（相关性 r=0.89）
2. 与 Felten et al. (2023) AI Occupational Exposure 指标对比

**一致性检查**：

- 理论上限 ≥ 近期分数
- 同行业相似职业分数差距 ≤ 2 分
- 高分职业必须有明确的 `tasks_at_risk` 列表

## 📊 数据统计

### 职业覆盖

- **总职业数**：42个代表性职业
- **覆盖就业**：约4.2亿人
- **行业分布**：
  - 信息技术：5个职业
  - 金融财务：4个职业
  - 制造生产：6个职业
  - 服务业：12个职业
  - 医疗健康：4个职业
  - 教育科研：5个职业
  - 其他：6个职业

### AI 影响分布


| 风险等级 | 分数区间  | 职业数 | 代表职业           |
| ---- | ----- | --- | -------------- |
| 极高风险 | 8-10分 | 4个  | 客服、会计、翻译、数据录入员 |
| 高风险  | 6-7分  | 12个 | 记者、营销、初级程序员    |
| 中风险  | 4-5分  | 15个 | 教师、护士、软件工程师    |
| 低风险  | 2-3分  | 8个  | 快递员、厨师、建筑工人    |
| 极低风险 | 0-1分  | 3个  | 家政服务员、美发师      |


### Impact Mode 分布

- **替代型 (Displacement)**：15% - 客服、翻译、数据录入
- **增强型 (Augmentation)**：45% - 程序员、设计师、营销
- **混合型 (Hybrid)**：30% - 会计、记者、教师
- **对抗型 (Resistance)**：10% - 法官、医生、律师
